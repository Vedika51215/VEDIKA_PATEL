Welcome! This is the official repository for my personal portfolio. It is designed around a **Stranger Things cinematic / retro-cyber theme**—bringing the "Upside Down" of the web directly to your browser. 

The site features custom 3D particle simulations, glowing retro typography, VHS scanline filters, and an interactive command shell.

---

## 🎨 Theme & Highlights

*   **Interactive Cyber Lab Terminal**: Run real-time mock hacker commands (e.g., `help`, `about`, `skills`, `certificates`, `nmap`) directly inside your browser to scan ports and audit capabilities.
*   **Cinematic Design**: Styled with a dark glassmorphic UI, CRT scanlines, and the iconic glowing red *Stranger Things* title font.
*   **3D Atmosphere**: A custom-built Three.js particle field simulation simulating floating spores, reflecting the Upside Down ambiance.
*   **Micro-Animations**: Custom magnetic spring cursor physics (automatically hides on mobile devices to prevent touch glitches) and smooth fade-in reveal animations powered by Framer Motion.
*   **Fully Responsive**: Hand-crafted layouts optimized for all viewport sizes (desktop, tablet, and mobile).

## 💻 Running the Project Locally

To run the Next.js portfolio website on your local machine, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (v18.x or above recommended).

### Steps

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Vedika51215/VEDIKA_PATEL.git
    cd VEDIKA_PATEL
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open in your browser**:
    Navigate to [http://localhost:3000](http://localhost:3000) to view the live rendering.
